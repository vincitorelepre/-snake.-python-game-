# Змейка на Python

Классическая игра "Змейка", реализованная на Python с использованием библиотеки Pygame. Проект представляет собой хорошо структурированную объектно-ориентированную реализацию с расширенным функционалом и подробными комментариями в коде.


## Возможности

- Классический геймплей с современными улучшениями
- Несколько уровней сложности
- Система бонусов и ускорения
- Таблица рекордов
- Кроссплатформенность
- Чистый и хорошо документированный код
- Подробные инструкции по установке и запуску

## Установка и запуск

### Требования
- Python 3.8+
- Pygame 2.0+

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск игры
```bash
python snake_game.py
```

## Управление
- **Стрелки** - движение змейки
- **R** - перезапуск игры
- **Q** - выход из игры
- **ESC** - выход

## Технические детали

### Основные классы
- **Snake** - управление змейкой и ее поведением
- **Food** - генерация и отрисовка еды
- **Game** - основной игровой цикл и логика

### Ключевые методы
- `Snake.move()` - движение змейки
- `Snake.turn()` - изменение направления
- `Game.update()` - обновление игрового состояния
- `Game.handle_events()` - обработка ввода
